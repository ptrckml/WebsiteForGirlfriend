# Miya's website

A simple personal website for journalism, documenting and thoughts.  The website has a soft lavender layout.

## Run it 

```powershell
python server.py
```
## Alternitively

```Go to where the file is located
Run server.py```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000).

## How it works

- Users can only browse every page and read posts.
- Only you can create or edit posts after logging in with the owner password.
- The first time you click `Owner Access`, the site asks you to create that password.
